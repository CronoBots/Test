✦ Admin — Photos

Remplace les photos du site Elaudace. Chaque photo est automatiquement convertie en JPEG + WebP et publiée sur GitHub. Le site se met à jour en ~1 minute.

1 · Connexion

Crée un jeton « fine-grained » sur github.com/settings/personal-access-tokens → Repository access : Only select repositories → CronoBots/ELAUDACE → Permissions → Contents : Read and write. Colle-le ci-dessus.

2 · Photos

🔒 Le jeton est stocké uniquement dans le navigateur de cet appareil (localStorage), jamais envoyé ailleurs qu'à GitHub. Tu peux le révoquer à tout moment depuis tes réglages GitHub. Cette page n'est pas indexée par les moteurs de recherche.
